# Harsha Vardan Gowda — Portfolio

Personal portfolio website for **Harsha Vardan Gowda**, a full-stack web developer for freelancers and agencies — building web apps, websites, and REST APIs (Django, Python), white-label ready.

🔗 **Live site:** _add your Vercel URL here once deployed_

## Tech

- Static **HTML / CSS / JavaScript** (no build step)
- Custom styling in `style.css`
- Interactions in `script.js`

## Project structure

```
.
├── index.html   # Main page
├── style.css    # Styles
└── script.js    # Interactions
```

## Run locally

Just open `index.html` in your browser, or serve it locally:

```bash
npx serve .
```

## Deployment

Hosted on **Vercel**. Connected to this GitHub repo — every push to `main` triggers an automatic redeploy.
